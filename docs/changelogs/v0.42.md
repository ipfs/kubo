#### ✨ New `ipfs cid inspect` command

A new `ipfs cid inspect <cid>` subcommand has been added to display detailed CID information including version, multibase encoding, multicodec, and multihash details (algorithm, length, digest). It also shows equivalent CIDv0/CIDv1 representations when applicable. Supports `--enc=json` for machine-readable output and works offline (no daemon required). See [#11205](https://github.com/ipfs/kubo/issues/11205).
